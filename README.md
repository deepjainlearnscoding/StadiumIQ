# 🏟️ CrowdPulse — Smart Stadium Intelligence Platform

CrowdPulse is a **real-time crowd intelligence and stadium navigation platform** designed to enhance fan experience, improve safety, and enable smarter decision-making during live events.

---

## 🚀 Live Demo

🔗 https://stadiumiq-224869760685.us-central1.run.app

---

## 📌 Problem Statement

Large stadiums often face challenges such as:

* Overcrowding at entry/exit gates
* Poor crowd distribution
* Inefficient navigation inside venues
* Delayed emergency response
* Lack of real-time operational insights

These issues directly impact **fan experience, safety, and stadium efficiency**.

---

## 💡 Solution

CrowdPulse provides a **data-driven interactive platform** that helps:

* Monitor crowd density through visual heatmaps
* Navigate efficiently داخل large stadiums
* Identify congestion hotspots in real-time
* Improve crowd flow and safety measures

---

## ✨ Key Features

* 📊 **Crowd Heatmap Visualization**
  Interactive heatmaps showing crowd density across different sections

* 🧭 **Smart Navigation System**
  Helps users find optimal paths inside the stadium

* 🚪 **Congestion Awareness**
  Highlights high-traffic zones and safer alternatives

* ⏱️ **Real-Time Simulation**
  Dynamic UI updates to mimic live crowd data

* 🔐 **Authentication System**
  Login/signup functionality for personalized access

* 🎨 **Modern UI/UX**
  Clean, responsive, and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Deployment:**

  * Google Cloud Run (Backend hosting)
  * Vercel (Frontend hosting)
* **Version Control:** Git & GitHub

---

## 🧠 How It Works

1. Simulated data represents crowd movement داخل the stadium
2. Data is visualized using heatmaps and interactive UI components
3. Users analyze congestion levels and plan movement accordingly
4. Navigation system guides users through less crowded routes
